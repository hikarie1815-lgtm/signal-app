# 建設レンタル・廃棄物処理管理アプリ

造園・建設業向けの、機械レンタル（開始・返却・延長）と廃棄物搬出・処分の記録管理アプリです。
**建設現場の従業員が iPhone / iPad から説明書なしで短時間入力できること**を最優先に設計しています。

設計ドキュメント（要件定義・画面一覧・権限表・DB設計・料金計算仕様・取込仕様・画面遷移・実装順序）は
[docs/SPEC.md](docs/SPEC.md) にあります。

## 起動方法

```bash
pip install -r requirements.txt
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

ブラウザ（iPhone / iPad は Safari）で `http://<サーバー>:8000/` を開きます。

1. 初回アクセスで「はじめての設定」画面が出るので、管理者名・会社名・メールアドレス・パスワードを設定
2. 管理者画面「従業員」タブから従業員2名を登録（ログインID＋仮パスワード）
3. 従業員は初回ログイン時に仮パスワードを変更
4. 管理者画面「単価表取込」で見積内訳書2枚（`seed/` に同梱、初回起動時に自動投入）の
   読取結果41行を原本画像と見比べて確認・承認すると料金マスターに本登録されます。
   原本で「環境対策費／日」が空欄の行は「要確認」のままです（0円なら0を入力して承認）

## ホーム画面に追加（アプリとして使う）

iPhone / iPad の Safari でアプリを開き、共有ボタン →「ホーム画面に追加」を選ぶと、
専用アイコン（青地のユンボ）でアプリのように起動できます（PWA対応・全画面表示）。

## テスト

```bash
python -m pytest tests/ -v
```

- `tests/test_pricing.py` — 料金計算モジュールの単体テスト（日割/月極/基本料/月跨ぎ配賦など）
- `tests/test_api.py` — 認証・権限・現場・レンタル・廃棄物・取込・集計・出力のAPIテスト

## 主な仕様

- 従業員は大ボタン6つのホーム画面＋1画面1作業のステップ入力（かんたん入力モード）
- 下書きは自動保存、通信が弱い現場では未送信データを端末に保持し復旧後に自動同期
- 料金は料金マスターから自動取得し、従業員は金額を編集できません（確認表示のみ）
- 従業員の削除は「取消申請」となり、管理者の承認で論理削除されます
- 単価表取込で読み取れない数値は補完せず「要確認」となり、承認できません
- 全操作は操作履歴（監査ログ）に記録され、管理者のみ閲覧できます
- 料金計算: 1〜9日は日割、10日以上は月極（30日ブロック）、基本料は開始月1回、
  サポート料・賠償対策費は毎日、整数円、月跨ぎは月別内訳を作成

## 技術構成

- バックエンド: Python / FastAPI / SQLite（`data.db`）
- フロントエンド: 依存ライブラリなしの HTML + JavaScript（スマホ最優先）
- Excel出力: openpyxl ／ PDF出力: 印刷用HTML（ブラウザの「PDFとして保存」）
