# SIGNAL DESK — リアルタイム売買シグナルアプリ(MVP)

FX・ゴールド・仮想通貨を監視し、AIスコア(0〜100点)で BUY / SELL / WAIT を判定する
**売買判断補助ツール**です。スマホのブラウザで見られます。

> ⚠️ 本アプリは分析ツールであり、投資助言ではありません。自動売買は行いません。
> 最終的な投資判断はご自身の責任で行ってください。

---

## 1. できること

- リアルタイム価格(仮想通貨は秒単位 / FX・金は無料枠では約2分間隔のローテーション更新)
- AIスコア 0〜100点 → STRONG BUY / BUY / WAIT / SELL / STRONG SELL
- 勝率・Profit Factor・期待値(直近データの簡易バックテストから自動計算)
- 推奨損切り(SL)・推奨利確(TP)・リスクリワード・推奨ロット(資金100万円・1%ルール)
- 総合 / 買い期待値 / 売り期待値 / 強シグナルのみ のランキング表示
- 銘柄詳細: チャート・シグナル履歴・エントリー理由・回避理由・上位足判定・最大DD
- バックテスト(1分/5分/15分/1時間足)+ トレード履歴CSVダウンロード
- 通知: Discord / LINE / ブラウザ通知(STRONGシグナル・高勝率シグナル・損失警告・連敗警告)
- 1日最大損失3万円で警告 / 連敗時に取引停止警告(損益を手動記録)

## 2. インストール手順

### 必要なもの
- Python 3.10以上(https://www.python.org/ からインストール。
  Windowsはインストール時に **「Add Python to PATH」に必ずチェック**)
- Twelve Data の無料APIキー(FX・ゴールド用。仮想通貨だけならキー不要で動きます)

### Windows
1. このフォルダを好きな場所に置く(例: `C:\signal-app`)
2. `start_windows.bat` をダブルクリック
3. 初回起動時に `.env` が自動作成される → メモ帳で開いて
   `TWELVEDATA_API_KEY=ここにキー` を貼り付けて保存
4. もう一度 `start_windows.bat` を実行
5. ブラウザで **http://localhost:8000** を開く

### Mac
```bash
cd signal-app
bash start_mac.sh
```

### スマホで見る方法
PCとスマホを同じWi-Fiに接続し、PCのIPアドレス(例: 192.168.1.10)を調べて
スマホのブラウザで `http://192.168.1.10:8000` を開きます。
- WindowsのIP確認: コマンドプロンプトで `ipconfig` → 「IPv4 アドレス」
- 初回はWindowsのファイアウォール許可ダイアログで「許可」を選ぶ

## 3. API設定方法

### Twelve Data(FX・ゴールド・シルバー)— 必須
1. https://twelvedata.com/ で無料登録
2. ダッシュボードの「API Key」をコピー
3. `.env` の `TWELVEDATA_API_KEY=` に貼り付け

### Discord通知(任意)
1. Discordのサーバー → チャンネル設定 → 連携サービス → ウェブフック → 新しいウェブフック
2. 「ウェブフックURLをコピー」→ `.env` の `DISCORD_WEBHOOK_URL=` に貼り付け

### LINE通知(任意)
1. https://developers.line.biz/ でMessaging APIチャンネルを作成
2. 「チャネルアクセストークン(長期)」を発行 → `LINE_CHANNEL_ACCESS_TOKEN=`
3. チャネル基本設定の「あなたのユーザーID」→ `LINE_USER_ID=`
4. 作成したBotを自分のLINEで友だち追加

### ブラウザ通知
画面右上の 🔔 ボタンを押して「許可」を選ぶだけ。

## 4. 無料で使える範囲(データAPI比較)

| API | 対象 | 無料枠 | 秒単位 | WebSocket | 判定 |
|---|---|---|---|---|---|
| **Binance 公開API** | 仮想通貨 | キー不要・実質無制限 | ◎ | ◎ 無料 | **採用(仮想通貨)** |
| Bybit 公開API | 仮想通貨 | キー不要 | ◎ | ◎ 無料 | 予備候補 |
| CoinGecko | 仮想通貨 | 30回/分 | ✕(遅延あり) | ✕ | 不採用 |
| **Twelve Data** | FX・金・銀 | 8回/分・800回/日 | ✕(ローテで約2分間隔) | 有料のみ | **採用(FX/金)** |
| Alpha Vantage | FX | 25回/**日** | ✕ | ✕ | 不採用(少なすぎ) |
| FMP | FX・指数 | 250回/日 | ✕ | ✕ | 予備候補 |
| OANDA(海外demo) | FX・金・指数 | demo口座で無料 | ◎ ストリーミング | ◎相当 | **次の本命** |

ポイント:
- **仮想通貨4銘柄は完全無料で秒単位** に動きます(Binance WebSocket)。
- **FX・金は無料枠の制約上、12銘柄を順番に更新**するため1銘柄あたり約2分間隔です。
  シグナルは5分足ベースなので実用上は十分ですが、「FXも秒単位」にしたい場合は下の有料候補へ。
- 株価指数(NAS100/US30/SPX500)と原油(USOIL)はTwelve Dataの**有料プラン(Grow)が必要**
  なため初期はOFFです。`.env` の `ENABLE_INDICES=true` で有効化できます。

## 7. AIスコアの仕組み(0〜100点)

| 評価項目 | 配点 |
|---|---|
| EMA 9/21/50/200 の並びと位置 | ±20 |
| MACD(クロス+ヒストグラム方向) | ±15 |
| 上位足一致(15分足±7・1時間足±8) | ±15 |
| RSI 14(極端値はペナルティ) | ±10 |
| ADX/DI(トレンドの強さと方向) | ±10 |
| ボリンジャーバンド内の位置 | ±10 |
| サポート/レジスタンスとの距離 | ±10 |
| 出来高(平均比) | ±5 |
| 直近20本高値安値ブレイク | ±5 |

判定: 85〜100 STRONG BUY / 70〜84 BUY / 40〜69 WAIT / 20〜39 SELL / 0〜19 STRONG SELL
フィルター: ADX低下+バンド収縮で**レンジ判定**しスコアを50へ圧縮(ダマシ除外)、ATR急拡大で高ボラ警告。

## 8. よくあるトラブル

| 症状 | 対処 |
|---|---|
| `python が見つかりません` | Pythonインストール時に「Add to PATH」を有効にして再インストール |
| FX銘柄が「APIキー未設定」 | `.env` に `TWELVEDATA_API_KEY` を設定して再起動 |
| 仮想通貨が出ない | ネット接続・ウイルス対策ソフトの通信ブロックを確認 |
| スマホから開けない | 同じWi-Fiか確認、ファイアウォールで8000番を許可 |
| 統計が「計算中」のまま | 起動から1〜2分待つ(データ蓄積後に自動計算) |
